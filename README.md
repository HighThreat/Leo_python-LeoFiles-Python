# Leo Python (Scripts & Utilities)

Colección de scripts y utilidades en Python enfocadas en automatización, lógica y operaciones.

## 🚀 Objetivo
Este repositorio consolida pequeñas herramientas y ejercicios desarrollados en Python, demostrando mi manejo del lenguaje, estructuración de código y resolución de problemas lógicos.

## 📁 Estructura del Repositorio

- `scripts/magic_8_ball.py`: Implementación lógica simple usando librerías estándar (`random`).
- `scripts/calculadora_moralidad.py`: Analizador de decisiones basado en marcos filosóficos éticos (CLI interactivo).
- `scripts/decision_tree.py`: Ejercicio de validación de requisitos mediante estructuras de control.

## 🛠️ Tecnologías
- **Python 3.x**
- Librerías estándar (no requiere dependencias externas).

## ⚙️ Uso Básico

Clona el repositorio y ejecuta cualquiera de los scripts directamente:

```bash
python scripts/calculadora_moralidad.py
```
